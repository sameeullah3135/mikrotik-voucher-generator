🎫 MikroTik Voucher Generator

Multi-router voucher management system for Hotspot

🔄 Checking API Status...

✨ Features:

  • Multi-router support (v6 & v7)
  • JWT Authentication with roles
  • Bulk voucher generation
  • PDF/CSV export
  • Printable templates (Thermal/A4)
  • Encrypted credentials in MongoDB
  • Voucher history & tracking
📡 API Endpoints:
POST /api/auth/login - Login
GET /api/routers - List routers
POST /api/vouchers/generate - Generate vouchers
GET /api/vouchers/export/csv - Export to CSV
